About The Rickshaw Run

A 3,500km pan-Indian adventure in a 7 horsepower glorified lawnmower. No set route, no back-up, no way of knowing if you're going to make it. The only certainty is that you will get lost, you will get stuck and you will break down. More info here.

About Us

One Spaniard, one Kiwi and one Portuguese (this is not a joke) decided to take on a journey in a wholly unsuitable vehicle, traversing the subcontinent enduring whatever shit the road throws at us. Great idea! Uh?

About the Funraising

This journey is also about helping to save the world. As part of the adventure we propose ourselves to raise some money for the Cool Earth Charity, the official Rickshaw Run charity.

Donate and leave a message

For that purpose we have a plain painted Rickshaw waiting for your message.
